
# Palo Alto Firewall – README

## 1. Introduction
This README provides an overview of Palo Alto Networks Firewalls, key features, and hands-on labs for learning and practicing firewall configurations. It is designed for beginners as well as professionals who want structured learning material.

## 2. Information
Palo Alto Networks Next-Generation Firewalls (NGFW) provide advanced security capabilities including:
- App-ID for application-based control
- User-ID for user-based visibility
- Content-ID for threat prevention
- URL filtering and DNS security
- WildFire cloud-based malware analysis
- Centralized management via Panorama

Key benefits include:
- Granular visibility into network traffic
- Zero Trust architecture implementation
- High availability options
- Scalable security across data centers and cloud environments

## 3. Labs
Below are sample labs to help learners practice key concepts:

### **Lab 1 – Basic Device Setup**
- Configure management interface
- Set hostname and DNS
- Add administrator accounts
- Configure NTP

### **Lab 2 – Interface & Zone Configuration**
- Create zones
- Configure Layer 3 interfaces
- Create VLAN and tunnel interfaces

### **Lab 3 – Security Policies**
- Create security rules
- Understand policy evaluation order
- Enable logging and log forwarding

### **Lab 4 – NAT Policies**
- Configure source NAT
- Configure destination NAT
- Test with client systems

### **Lab 5 – App-ID & User-ID**
- Enable App-ID in policies
- Integrate Active Directory for User-ID
- Test visibility in logs

### **Lab 6 – Threat Prevention**
- Configure Antivirus, Anti-Spyware, and Vulnerability Protection
- Attach profiles to security rules
- Verify results via test tools

### **Lab 7 – Panorama Management**
- Add firewall to Panorama
- Push templates and device groups
- Manage logs centrally
